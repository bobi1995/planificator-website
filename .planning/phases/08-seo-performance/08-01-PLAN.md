  <action>
    1. Install packages: `npm install next-plausible schema-dts`

    2. Create `src/lib/constants.ts`:
       - Export `SITE_URL` that reads from `process.env.NEXT_PUBLIC_SITE_URL` with fallback to `'https://planifactor.com'`
       - Export `SITE_NAME = 'Planifactor'`

    3. Create `src/lib/metadata.ts`:
       - Import `SITE_URL` from constants
       - Import `routing` from `@/i18n/routing`
       - Export `buildAlternates(locale: string, path: string)` function that returns the `alternates` object for generateMetadata:
         ```
         {
           canonical: `${SITE_URL}/${locale}${path}`,
           languages: Object.fromEntries(
             routing.locales.map((l) => [l, `${SITE_URL}/${l}${path}`])
           ),
         }
         ```
       - The `path` parameter should include leading slash (e.g., '/features') or be empty string for home page

    4. Create `src/lib/structured-data.ts`:
       - Import `Organization, SoftwareApplication, Article, WithContext` from `schema-dts`
       - Import `SITE_URL, SITE_NAME` from constants
       - Export `OrganizationJsonLd` React server component that renders a `<script type="application/ld+json">` with Organization schema:
         - name: SITE_NAME, url: SITE_URL, logo: `${SITE_URL}/icon.svg`
         - description: 'AI-powered production scheduling platform for manufacturers'
         - MUST sanitize output: `JSON.stringify(jsonLd).replace(/</g, '\\u003c')` to prevent XSS
       - Export `SoftwareApplicationJsonLd` server component for the landing page:
         - @type: SoftwareApplication, name: SITE_NAME, applicationCategory: 'BusinessApplication'
         - operatingSystem: 'Web', offers: { @type: Offer, price: '0', priceCurrency: 'EUR', description: 'Contact for pricing' }
       - Export `ArticleJsonLd({ title, description, datePublished, author, url })` server component for blog posts:
         - @type: Article, headline: title, description, datePublished, author: { @type: Person, name: author }
         - publisher: { @type: Organization, name: SITE_NAME, url: SITE_URL }
       - All three components use `dangerouslySetInnerHTML` with XSS-safe `.replace(/</g, '\\u003c')`

    5. Create `src/lib/og-image.tsx`:
       - Export `createOgImageElement({ title, subtitle }: { title: string; subtitle?: string })` that returns JSX element (NOT a React component -- just the JSX for ImageResponse)
       - Design: full-bleed gradient background `linear-gradient(135deg, #1e3a8a 0%, #2563eb 60%, #3b82f6 100%)`, white text
       - Top-left: "Planifactor" text (fontSize 28, fontWeight 600, opacity 0.8) positioned with `position: 'absolute', top: 60, left: 80` and `display: 'flex'` (satori requires display flex on every element)
       - Bottom-left: title (fontSize 56, fontWeight 800, lineHeight 1.15) and optional subtitle (fontSize 28, fontWeight 400, opacity 0.85)
       - Main container: `alignItems: 'flex-start', justifyContent: 'flex-end', padding: '60px 80px'`
       - IMPORTANT: Only use flexbox layout. No CSS grid. No gradients on text. This is rendered by satori.
       - Export `OG_SIZE = { width: 1200, height: 630 }` and `OG_CONTENT_TYPE = 'image/png'` constants

    6. Create `src/lib/analytics.ts`:
       - Export `PlausibleEvents` type mapping custom event names to their property types:
         ```typescript
         export type PlausibleEvents = {
           'Demo Request Click': { location: string };
           'ROI Calculator Used': { estimatedSavings: number };
           'CTA Click': { variant: string; page: string };
           'Blog Post Read': { slug: string };
         };
         ```
       - This is a types-only file. No runtime code.
  </action>

    - All modules compile with zero type errors
  </done>

       - Import `SITE_URL` from `@/lib/constants`
       - Export default async function returning `Promise<MetadataRoute.Sitemap>`
       - Static pages array: `['', '/features', '/use-cases', '/pricing', '/about', '/blog', '/contact', '/roi-calculator', '/privacy', '/terms', '/use-cases/discrete-manufacturing', '/use-cases/job-shops', '/use-cases/production-planning']`
       - For each static page, create entry with:
         - `url: ${SITE_URL}/en${path}` (canonical is EN)
         - `lastModified: new Date()`
         - `alternates.languages`: Object.fromEntries mapping each locale to its full URL
       - For blog posts: call `getAllPosts('en')` and create entries with:
         - `url: ${SITE_URL}/en/blog/${post.slug}`
         - `lastModified: new Date(post.meta.date)`
         - Same alternates pattern as static pages
       - Return spread of static + blog entries
       - IMPORTANT: This file is at `src/app/sitemap.ts` (app root), NOT inside `[locale]`. It generates `/sitemap.xml` at domain root.

    2. Create `src/app/robots.ts`:
       - Import `MetadataRoute` from `next`
       - Import `SITE_URL` from `@/lib/constants`
       - Export default function returning `MetadataRoute.Robots`
       - Single rule: `{ userAgent: '*', allow: '/' }`
       - `sitemap: ${SITE_URL}/sitemap.xml`
