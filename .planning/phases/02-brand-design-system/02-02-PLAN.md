<task type="auto">
  <name>Task 1: Create Logo component and favicon</name>
  <files>src/components/brand/Logo.tsx, src/app/icon.svg</files>
  <action>
    **Step 1: Create `src/components/brand/Logo.tsx`**

    Build an SVG logo React component with the following specs from CONTEXT.md:
    - Icon metaphor: Abstract Gantt bars / timeline -- 3-4 horizontal bars of varying lengths suggesting a schedule
    - Weight: Light and elegant, thin strokes (stroke-width 2-3px), breathing room between bars
    - The icon portion should fit in a square viewBox (e.g., 48x48)
    - The full logo (icon + "Planifactor" text) should have a wider viewBox (e.g., 240x48)

    Props interface:
    ```tsx
    type LogoVariant = "default" | "white" | "icon";
    type LogoSize = "xs" | "sm" | "md" | "lg";
    ```

    Size map (from research):
    - xs: icon 24px, full 120x24 (favicon context)
    - sm: icon 28px, full 140x28 (mobile header)
    - md: icon 36px, full 180x36 (desktop header)
    - lg: icon 48px, full 240x48 (footer / hero)

    Behavior:
    - `variant="default"`: uses `currentColor` for fills (inherits from parent text color -- near-black on white backgrounds)
    - `variant="white"`: uses `#ffffff` for fills (for use on blue backgrounds)
    - `variant="icon"`: renders only the Gantt bars icon, no text. Uses square viewBox.
    - Include `aria-label="Planifactor"` and `role="img"` for accessibility
    - Use `cn()` from `@/lib/utils` for className merging
    - The text "Planifactor" should use fontFamily `"var(--font-inter), Inter, sans-serif"`, fontWeight 600, positioned to the right of the icon with appropriate spacing

    SVG design guidance for the Gantt bars icon:
    - Use rounded rectangles (`<rect rx="...">`) not lines
    - 3-4 bars stacked vertically with gaps between them
    - Bars should have different lengths (short, long, medium, medium-long) to suggest a Gantt chart
    - Bars should be thin (height ~4-5px with ~4px gaps in a 48px square)
    - Bars should NOT all start at x=0 -- stagger some starts to suggest scheduling offset
    - Use `fill` not `stroke` for the bars (solid fills, not outlines)
    - Keep it simple and geometric -- this needs to be recognizable at 16px

    **Step 2: Create `src/app/icon.svg`**

    Extract the icon-only portion of the logo SVG (just the Gantt bars, no text) and save as `src/app/icon.svg`. This is the favicon. Next.js auto-detects this file.

    - ViewBox should be square (e.g., "0 0 48 48")
    - Use the brand blue color directly: `fill="oklch(0.546 0.245 262.881)"` or the hex equivalent `#2563eb`
    - Do NOT use `currentColor` in the favicon -- it needs a fixed blue color

    Also DELETE `src/app/favicon.ico` -- it takes precedence over icon.svg and would prevent the new favicon from showing.
  </action>

  </verify>
