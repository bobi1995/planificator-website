**Logo grid:**
```tsx
<div className="text-center mb-16">
  <p className="text-label uppercase tracking-wider text-muted-foreground mb-8">
    {t('trustedBy')}
  </p>
  <div className="flex flex-wrap justify-center items-center gap-8 md:gap-12">
    {[0, 1, 2, 3, 4, 5].map((i) => (
      <div key={i} className="text-lg font-semibold text-muted-foreground/50">
        {t(`companies.${i}`)}
      </div>
    ))}
  </div>
</div>
```

The company "logos" are rendered as styled text (font-semibold, muted color). This is the placeholder approach — real logos will replace these later. The muted/50 opacity makes them look like greyed-out partner logos (a common B2B SaaS pattern).

**Testimonial cards:**
```tsx
<div className="grid grid-cols-1 md:grid-cols-3 gap-8">
  {[0, 1, 2].map((i) => (
    <div key={i} className="rounded-lg border bg-card p-6">
      <blockquote className="text-foreground mb-6">
        &ldquo;{t(`testimonials.${i}.quote`)}&rdquo;
      </blockquote>
      <div className="flex items-center gap-3">
        <Avatar>
          <AvatarFallback className="bg-brand-100 text-brand-700 text-sm font-semibold">
            {/* Extract initials from the name */}
            {t(`testimonials.${i}.name`).split(' ').map((n: string) => n[0]).join('')}
          </AvatarFallback>
        </Avatar>
        <div>
          <p className="font-semibold text-sm">{t(`testimonials.${i}.name`)}</p>
          <p className="text-muted-foreground text-sm">
            {t(`testimonials.${i}.title`)}, {t(`testimonials.${i}.company`)}
          </p>
        </div>
      </div>
    </div>
  ))}
</div>
```

**Initials extraction:** For the Avatar fallback, extract initials from the person's name. For example, "Stefan Petrov" becomes "SP", "Maria Georgieva" becomes "MG". Since Bulgarian names use Cyrillic (e.g., "Stефан Петров"), the initials will be Cyrillic characters in BG locale — this is correct and expected.

**Imports:**
```typescript
import {getTranslations} from 'next-intl/server';
import {Avatar, AvatarFallback} from '@/components/ui/avatar';
```

**Export:** `export async function SocialProof() { ... }`
Use: `const t = await getTranslations('SocialProof');`

**Key details:**
- Testimonial cards: 3 columns on desktop (md:grid-cols-3), stacked on mobile
- Avatar uses AvatarFallback only (no AvatarImage) since these are placeholder testimonials
- The `bg-brand-100 text-brand-700` on AvatarFallback matches the brand color palette
- Blockquote uses curly quotes via `&ldquo;` and `&rdquo;` HTML entities
</instructions>

**Part A: Rewrite page.tsx**

Replace the content of `src/app/[locale]/page.tsx` with the full landing page composition. The page imports and renders all 4 section components in order.

```typescript
import {setRequestLocale, getTranslations} from 'next-intl/server';
import {Hero} from '@/components/sections/Hero';
import {FeatureHighlights} from '@/components/sections/FeatureHighlights';
import {SocialProof} from '@/components/sections/SocialProof';
import {CTABanner} from '@/components/sections/CTABanner';

type Props = {
  params: Promise<{locale: string}>;
};

export async function generateMetadata({params}: Props) {
  const {locale} = await params;
  const t = await getTranslations({locale, namespace: 'Metadata'});

  return {
    title: t('title'),
    description: t('description'),
  };
}

export default async function HomePage({params}: Props) {
  const {locale} = await params;
  setRequestLocale(locale);

  return (
    <>
      <Hero />
      <FeatureHighlights />
      <SocialProof />
      <CTABanner />
    </>
  );
}
```

**Key changes from current page.tsx:**
- Remove Logo, Button imports (moved to section components)

- Remove the old centered layout with Logo and buttons
- Import and render section components in sequence
- Keep `generateMetadata` using the `Metadata` namespace (unchanged)
- Keep `setRequestLocale` call for static rendering
